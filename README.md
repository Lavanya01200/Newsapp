

# NewsRift

## Overview 
"Newrift," a ReactJS-powered web app for dynamic news aggregation with country-specific filtering, offering users a tailored, immersive news experience and direct access to source websites. Demonstrates proficiency in ReactJS, API integration, and user-focused design.


## Features
* Real-time aggregation of news content from diverse sources.
* Intelligent categorization into technology, business, health, and sports.
* Customizable filtering based on country codes for region-specific news.
* User-friendly presentation through a visually appealing card layout.
* "Read More" button for direct redirection to the original news source.
* Utilization of ReactJS for a dynamic and responsive user interface.
* Seamless integration with external APIs for real-time data retrieval.
* Thoughtful design choices for an enhanced user experience.
